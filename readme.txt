== Description ==

This plugin pays tribute to Galaktion Tabidze (1892–1959), one of Georgia's greatest poets. When activated, a randomly chosen line from his celebrated poem <cite>მთაწმინდის მთვარე</cite> (The Moon of Mtatsminda) will appear in the upper right of your WordPress admin screen on every page.

== Installation ==

1. Upload the `galaktion-tabidze` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Enjoy a line of Georgian poetry on every admin page.

== Frequently Asked Questions ==

= What language is the poem in? =

The poem is in Georgian (ქართული). Galaktion Tabidze wrote in Georgian and the lyrics are displayed as he wrote them.

= Can I translate the plugin interface? =

Yes. The plugin is fully internationalised. You can contribute translations at translate.wordpress.org.
